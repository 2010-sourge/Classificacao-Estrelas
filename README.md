# Avaliação 5 Estrelas

Projeto de prática de front-end que consiste em um **componente de avaliação por estrelas (star rating)**, desenvolvido com HTML, CSS.

## 📋 Sobre o projeto

Uma página simples que permite ao usuário selecionar uma nota de 1 a 5 estrelas, exibindo uma mensagem de feedback correspondente à avaliação escolhida.

## 🖼️ Preview

A página exibe:

- **Componente de 5 estrelas:** clicáveis/selecionáveis
- **Mensagem de feedback dinâmica:** exibida após a avaliação (ex.: "I love it 😍" para a nota máxima)

## 🛠️ Tecnologias utilizadas
